About the algorithm

This is what it does.

Introduction

Recommender systems are a type of machine learning algorithm that provides consumers with "relevant" recommendations. When we search for something anywhere, be it in an app or in our search engine, this recommender system is used to provide us with relevant results. They use a class of algorithms to find out the relevant recommendation for the user.

Our Recommendation System

RecomMed App uses a recommendation system based on keywords extraction and keyword matching between your input (symtoms, worries, curiosities) and a database of medical articles!

Technologies and APIs used

To build this algorithm integrated in the RecomMed App, we used Python as main programming language for the following tasks:

  • url extraction
  • text content
  • keyword extraction
  • summary extraction

All the above tasks were executed one time, from which we obtained a data base of 7588 medical articles matched with their relevant keywords to recommend them to you, and their summary to display them to you as a preview.

Furthermore, if we catch your interest, please - access the link to read the full article written by the Health A-Z team of doctors.

Conclusions

By typing your symptoms or your medical curiosities in the textarea offered by RecomMed App, then we can provide you with some medical articles based on your input.
